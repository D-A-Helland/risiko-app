# Teknisk dokumentasjon

## Stack

- Next.js (React framework)
- React (UI library)
- Tailwind CSS (Styling)
- Node.js (Runtime)
- Vercel (Deployment)

## VSCode plugins

- Prettier
- ESLint
- Tailwind CSS IntelliSense
- ES7 React/Redux Snippets

## Kommandoer for innstallasjon

    npx create-next-app@latest my-app -
    Oppretter prosjektet

    cd my-app -
    Går inn i prosjektmappen

    npm run dev -
    Starter utviklingsserveren
